# NEOCRAFT — Modern Voxel Sandbox

クールでモダンな雰囲気のマインクラフト風ボクセルサンドボックスゲーム。
ブラウザだけで動きます（Three.js / 単一HTMLファイル）。

## 遊び方

ローカルサーバーを立てて `index.html` を開きます。

```sh
python -m http.server 8765
# → http://localhost:8765 をブラウザで開く
```

## 操作

| キー | 動作 |
|---|---|
| W A S D | 移動 |
| マウス | 視点 |
| Space | ジャンプ / （水中・飛行中）上昇 |
| Shift | ダッシュ |
| 左クリック | ブロック破壊 |
| 右クリック | ブロック設置 |
| ホイール中クリック | ブロックスポイト |
| 1–9 / ホイール | ホットバー選択 |
| F | 飛行モード切替（C で下降） |
| Esc | メニュー |

## 特徴

- **プロシージャル地形生成** — ノイズベースの山・海・砂浜・雪山・森
- **昼夜サイクル** — 朝焼け・夕暮れ・星空がシームレスに変化（1周4分）
- **モダンなビジュアル** — アンビエントオクルージョン、ACESトーンマッピング、フォグ、流れる雲
- **ネオン演出** — 夜に光るグロウストーン、シアン×バイオレットのガラスモーフィズムHUD
- **サンドボックス** — 9種のブロックを設置/破壊、水泳・飛行モード、破壊パーティクル＆効果音

## 技術

- [Three.js](https://threejs.org/) r160（CDN読み込み、ビルド不要）
- チャンクベースのメッシュ生成（面カリング＋頂点AO）
- 依存パッケージなし・単一HTMLファイル
